# OpenCV-Aimbot

## Overview  
This script is an aim assistance tool that uses computer vision (template matching) to detect targets and automatically move the mouse cursor when aiming.

## Instructions
Run `aimbot.py`  
`&lt;RMB&gt;` - Hold to aim (when active)  
`&lt;Up&gt;` - Toggle character aim on/off  
`&lt;Down&gt;` - Exit aimbot  

## Dependencies  
Required libraries: pywin32, numpy, opencv-python, mss  
Install with bash: pip install pywin32 numpy opencv-python mss
